# 2026-07-15 개발 이력

## 작업 내용

### 미도입 전략 리서치 문서(2026-07-14) 기반 7개 기능 구현

어제 작성한 갭 분석 리서치 문서에서 도출된 미구현 전략을 우선순위 순서대로 전부 구현했다.

---

### 1. RSI + Bollinger Bands (utils/indicators.py)

- calculate_all()에 RSI14·BB(20,2σ)·ATR14 추가 (기존 정적 메서드 활용)
- df["RSI14"], df["BB_upper"], df["BB_lower"], df["ATR14"] 컬럼 생성
- 가격 차트에 BB 상단/하단 점선(청록색) 오버레이
- RSI14 서브차트(130px 높이) 추가 — 70/30 기준선 포함

---

### 2. ATR 기반 손절 (utils/strategy_engine.py)

- calculate_pullback_plan()에 atr14 파라미터 추가
- ATR14 제공 시: 손절 = 현재가 − 2.5×ATR14 (변동성 기반)
- ATR14 없을 시: fallback VWAP×0.96 (기존 방식)
- stop_loss_method 필드로 UI에 방식 명시
- State.selected_atr14 — 차트 로드 시 최신 ATR14 추출·저장

---

### 3. 52주 신고가 뱃지 (scanner.py, main.py)

- scanner.py: high_252 = Close.rolling(252, min_periods=60).max() 계산 → New52WHigh 필드
- ScanResult.new_52w_high: bool 필드 추가
- 스캔 결과 테이블에 "52W" 컬럼 추가 — 해당 종목에 52W↑ 녹색 배지 표시

---

### 4. VKOSPI 공포지수 (utils/data_loader.py, main.py)

- QuantDataLoader.get_vkospi() — FinanceDataReader로 VKOSPI 조회
- 레벨 분류: ≥25 공포(적색), 20~25 주의(주황), <20 안도(초록)
- 분석 탭 KR 종목에만 VKOSPI 배너 표시 (조회 버튼 → 비동기 로드)

---

### 5. 현실적 거래 비용 (backtester.py)

- KR: 거래세 0.18%(매도) + 수수료 0.015%(양방향) + 슬리피지 0.05%(양방향) = 왕복 ~0.33%
- US: 수수료 0.005%(양방향) + 슬리피지 0.05%(양방향) = 왕복 ~0.11%
- _simulate() 내 매수/매도에 비용 반영 (유효 매수가·실수령액 계산)
- BacktestSummary.cost_per_trade_pct 필드 추가, 백테스트 결과 카드에 표시

---

### 6. Kelly Criterion (utils/strategy_engine.py, main.py)

- calc_kelly_fraction(win_rate, avg_win_pct, avg_loss_pct) 함수 추가
- Half-Kelly 적용 (과베팅 방지) + 최대 25% 한도(_KELLY_MAX)
- calculate_pullback_plan()에 win_rate / avg_win_pct / avg_loss_pct 파라미터 추가
- 반환 dict에 kelly_fraction, kelly_budget 포함
- 분할 매수 플랜 패널에 "켈리 권장 투자 비율" 보라색 섹션 표시

---

### 7. 드로우다운 서킷브레이커 (main.py)

- 보유종목분석 탭 포트폴리오 pnl_pct 기준 3단계 경보
  - -10% → 주의(황색): 신규 매수 축소 권고
  - -15% → 경고(주황): 포지션 50% 축소 권고
  - -20% → 서킷브레이커(적색): 전체 청산/매수 중단 권고
- rx.callout 컴포넌트로 조건부 표시
