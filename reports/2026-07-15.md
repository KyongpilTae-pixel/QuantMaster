# 2026-07-15 개발 이력

## 작업 요약

미도입 전략 리서치 문서(2026-07-14) 기반으로 단기 7건 + 중기 4건, 총 **11개 기능** 구현.  
전략 리서치 문서 구현 현황 11/29 → 15/29 (37.9% → 51.7%) 동기화.

## 구현 내역

### 단기 (즉시 착수) — 7건

#### 1. RSI + Bollinger Bands (utils/indicators.py)

- `calculate_all()`에 RSI14 · BB(20일, 2σ) · ATR14 추가
- `df["RSI14"]`, `df["BB_upper"]`, `df["BB_lower"]`, `df["ATR14"]` 컬럼 생성
- 가격 차트에 BB 상단/하단 점선(청록색) 오버레이
- RSI14 서브차트(130px) 추가 — 과열(70) · 과매도(30) 기준선 포함

#### 2. ATR 기반 손절 (utils/strategy_engine.py)

- `calculate_pullback_plan()`에 `atr14` 파라미터 추가
- ATR14 제공 시: 손절선 = 현재가 − 2.5 × ATR14 (변동성 기반)
- ATR14 없을 시: 폴백 VWAP × 0.96 (고정 비율)
- `stop_loss_method` 필드로 UI에 산출 방식 명시
- `State.selected_atr14` — 차트 로드 시 최신 ATR14 추출·저장

#### 3. 52주 신고가 뱃지 (scanner.py, main.py)

- `high_252 = Close.rolling(252, min_periods=60).max()` → `New52WHigh` bool 필드
- 스캔 결과 테이블 "52W" 컬럼 추가 — 해당 종목에 `52W↑` 녹색 배지 표시

#### 4. VKOSPI 공포지수 (utils/data_loader.py, main.py)

- `QuantDataLoader.get_vkospi()` — FinanceDataReader로 VKOSPI 조회
- 레벨 분류: ≥25 공포(적색) / 20–25 주의(주황) / <20 안도(초록)
- 분석 탭 KR 종목에만 배너 표시 (조회 버튼 → 비동기 로드)

#### 5. 현실적 거래 비용 (backtester.py)

- KR: 거래세 0.18%(매도) + 수수료 0.015%(양방향) + 슬리피지 0.05%(양방향) = 왕복 ~0.33%
- US: 수수료 0.005%(양방향) + 슬리피지 0.05%(양방향) = 왕복 ~0.11%
- `_simulate()` 내 유효 매수가 · 실수령액 계산 반영
- 백테스트 결과 카드에 `왕복거래비용%` 표시

#### 6. Kelly Criterion 포지션 사이징 (utils/strategy_engine.py, main.py)

- `calc_kelly_fraction(win_rate, avg_win_pct, avg_loss_pct)` 함수 추가
- Half-Kelly 적용 (과베팅 방지) + 최대 25% 한도 (`_KELLY_MAX`)
- `calculate_pullback_plan()` 반환 dict에 `kelly_fraction`, `kelly_budget` 포함
- 분할 매수 플랜 패널에 "켈리 권장 투자 비율" 섹션 추가

#### 7. 드로우다운 서킷브레이커 (main.py)

- 보유종목분석 탭 포트폴리오 손익률 기준 3단계 경보
  - −10% → 주의(황색): 신규 매수 축소 권고
  - −15% → 경고(주황): 포지션 50% 축소 권고
  - −20% → 서킷브레이커(적색): 전체 청산 / 매수 중단 권고
- `rx.callout`으로 조건부 표시

### 중기 (1–2개월) — 4건

#### 8. Monte Carlo 시뮬레이션 (backtester.py, main.py)

- `Backtester.run_monte_carlo(trades, initial_capital, n_sim=1000)` 정적 메서드
- numpy 벡터 연산: `(n_sim × n_trades+1)` 자본금 행렬 구성 (시드 42 고정)
- 매 시뮬레이션마다 실제 수익률 복원 추출(bootstrap) → 자본금 곡선 계산
- 출력: p5/p50/p95 최종 자본, VaR5%, CVaR1%, 파산확률(초기자본 50% 미만)
- UI: "시뮬레이션 실행 (1,000회)" 버튼 + 6개 지표 카드 + area_chart 3 시리즈

#### 9. Risk Parity 배분 (utils/momentum_scanner.py, main.py)

- `fetch_momentum_data()`에 섹션 6 추가 — scipy SLSQP 공분산 최적화
- 목적함수: 각 자산 위험 기여도를 1/N으로 균등화 `Σ(RC_i − 1/N)²`
- scipy 미설치 / 수렴 실패 시 역변동성 배분으로 자동 폴백
- 글로벌 모멘텀 탭 "위험균형" 전략 카드 + 배분 테이블(violet 바 차트) 추가

#### 10. 배당 성장 스크리닝 (utils/dividend_scanner.py 신규, main.py)

- `scan_dividend_stocks(market, min_yield_pct, max_payout_pct, top_n)`
  - KR: `get_market_snapshot()` DivYield 컬럼 직접 필터 (즉시)
  - US: S&P500 200개 샘플 → yfinance 병렬 수집 (`max_workers=8`)
- KR 배당성향 간이 추정: `배당수익률% × PER / 100`
- 배당성장 여부: yfinance 연간 배당 이력 전년 대비 비교
- 스캔 모드 드롭다운 "배당 스캔" 추가, opts 패널(2%/3%/5%/7%), 결과 테이블

#### 11. 외국인·기관 순매수 추적 (utils/data_loader.py, main.py)

- `QuantDataLoader.get_investor_trading(symbol, lookback_days=20)`
  - pykrx `get_market_trading_volume_by_investor()` 우선 시도
  - pykrx 실패 시 NAVER Finance 투자자 동향 페이지 스크래핑 자동 폴백
  - 반환: `rows`(일별 외국인/기관/개인 순매수), `cumulative`(기간 누계)
- `investor_trading_panel()` — 분석 탭 KR 전용 노출
  - 누계 3개 배지 카드(외국인/기관/개인) + 일별 테이블(색상 인코딩)
