### 4. 주간·월간 리포트 — 시장 장세 판단 + 투자 전략 추천 섹션 신규 추가

#### 4-1. `utils/market_regime.py` (신규)

ADX/DMI 알고리즘으로 주요 4개 지수(KOSPI/KOSDAQ/S&P500/NASDAQ) 장세 자동 판별

**알고리즘**:
- ADX ≥ 25 = 추세장 (Wilder 14일 평활화)
- ADX ≥ 25 + +DI > -DI + 가격 > MA20 → **상승** (green)
- ADX ≥ 25 + -DI > +DI + 가격 < MA20 → **하락** (red)
- 위 조건 불충족 → **횡보** (gray)

**주요 함수**:
- `detect_regime(code, lookback=150)` → ADX/DI/MA20 상세 dict
- `fetch_all_regimes()` → 4개 지수 병렬 조회
- `generate_regime_section(generated_at, regimes)` → HTML 테이블 섹션

#### 4-2. `utils/regime_picks.py` (신규)

장세별 최적 스캔 전략 + 캐시 기반 TOP5 종목 추천

**전략 매핑**:
| 장세 | 전략 | 종목 선발 기준 |
|------|------|--------------|
| 상승 | 추세추종 (모멘텀 집중) | 모멘텀 캐시 3M 수익률 상위 |
| 횡보 | 눌림목 저점 매수 | 1W < -2% + 3M > 0% 필터 |
| 하락 | 하락방어 (Beta 낮음) | 3M > 0% + 1W 선방 순 |

- 캐시 기반 (`momentum_{market}_*.json`, `leaders_{market}_*.json`) — 네트워크 없이 즉시 생성
- `generate_regime_picks_section(regimes, generated_at)` → KR/US 블록 HTML

#### 4-3. 주간 리포트 (`utils/weekly_report_generator.py`) 수정
