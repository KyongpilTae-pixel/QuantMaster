GP/A — Novy-Marx 총수익성 팩터 즉시 구현 쉬움
매출총이익을 총자산으로 나눈 단순 수식. 한국 시장에서 CAGR 69%로 가장 강력한 단일 팩터. yfinance 데이터만으로 계산 가능하며 factor_loader.py에 즉시 추가 가능.
GP/A = (Revenue − COGS) / Total Assets
KOSPI CAGR 69% Novy-Marx 2013
강환국 슈퍼가치 — 4팩터 순위합산 즉시 구현 쉬움
PER·PBR·PCR·PSR 각각 낮은 순으로 순위를 매겨 합산. 순위합이 낮은 종목이 "싼 종목". NAVER Finance 데이터(이미 스크래핑 중)에서 PCR만 추가하면 됨.
Score = rank(PER) + rank(PBR) + rank(PCR) + rank(PSR) → 낮을수록 매수
KOSPI CAGR 44.7% 강환국 2023
강환국 슈퍼퀄리티 — GP/A + 신F-Score 단기 구현 중간
GP/A 상위 50% × Piotroski 신F-Score ≥ 7점 × 소형주(시총 하위 30%) 교집합. 질적 우량 소형주 발굴. F-Score는 이미 구현됐으므로 GP/A 추가 후 필터 체인 연결만 하면 됨.
GP/A ≥ 중간값 AND F-Score ≥ 7 AND 시총 소형주
KOSPI CAGR 46.1% 강환국 2023
기업밸류업 스크리너 — PBR + 자사주 단기 구현 쉬움
금융당국의 기업 밸류업 프로그램 참여 기업 선별. PBR < 1.0이면서 자사주 매입 증가 + ROE 개선 흐름. 정책 수혜 모멘텀 포착.
PBR < 1.0 AND △자사주 > 0 AND ROE(t) > ROE(t-1)

Altman Z''-Score — 비제조업 부도위험 즉시 구현 중간
5개 재무변수 조합으로 기업 부도 가능성을 정량화. Z'' 모델은 서비스업·비제조업에 적합(한국 주요 업종 포함). Z'' < 1.1이면 부도위험 구간 — 스캔 제외 필터로 즉시 활용 가능.
Z'' = 6.56·X1 + 3.26·X2 + 6.72·X3 + 1.05·X4
X1=운전자본/자산 X2=이익잉여금/자산
X3=EBIT/자산 X4=자기자본/총부채
구간Z'' 값판단
안전> 2.6부도 가능성 낮음
회색1.1 – 2.6주의 관찰 필요
위험< 1.1부도위험 — 스캔 제외 권장
예측 정확도 93% Altman 1995
Beneish M-Score — 분식회계 탐지 즉시 구현 중간
8개 재무지표 변화율 조합으로 이익 조작(earnings manipulation) 가능성을 측정. M > -2.22이면 분식 의심. 엔론·월드컴 등 사전 식별 사례. 한국 회계 스캔들 사전 방어에 효과적.
M = -4.84 + 0.92·DSRI + 0.528·GMI + 0.404·AQI
+ 0.892·SGI + 0.115·DEPI − 0.172·SGAI
+ 4.679·TATA − 0.327·LVGI
임계값: M < −2.22 → 조작 가능성 낮음 (안전)
임계값 −2.22 8-variable
이자보상배율 (ICR) — 이자 지급 능력 단기 구현 쉬움
영업이익으로 이자를 몇 배 갚을 수 있는지 측정. ICR < 1이면 영업이익으로 이자도 못 내는 '좀비기업'. 스캔 최소 조건으로 ICR ≥ 3 권장. yfinance 데이터 즉시 활용 가능.
ICR = EBIT / 이자비용 → ≥ 3 권장
권장 기준 ≥ 3x 안전마진
순부채 / EBITDA — 레버리지 비율 단기 구현 쉬움
순부채(총부채 − 현금)를 EBITDA로 나눈 비율. 부채 상환에 몇 년이 걸리는지 직관적 측정. 3배 이하 = 건전, 5배 초과 = 고레버리지 위험. M&A 실사에서도 핵심 지표.
순부채/EBITDA = (총부채 − 현금) / EBITDA → ≤ 3 권장
위험 기준 ≥ 5x 고레버리지

FCF Yield — 잉여현금흐름 수익률 단기 구현 쉬움
기업이 실제로 창출하는 현금을 EV 대비로 본 수익률. PER보다 분식 조작이 어렵고 장기 수익률 예측력이 높음. factor_loader.py의 load_value_metrics()에 FCF/EV 필드 추가로 즉시 구현 가능.
FCF Yield = Free Cash Flow / Enterprise Value
FCF = 영업현금흐름 − CapEx
40yr CAGR 16.6% S&P500 기준
현금전환주기 (CCC) — 운전자본 효율 중기 구현 중간
재고→판매→대금 수취→구매대금 지급까지의 날수. 낮을수록 현금 효율이 좋음. 동종업 평균 대비 CCC 개선 추세가 의미 있음. 산업별 벤치마크 비교 필요.
CCC = DIO + DSO − DPO
DIO = 재고/매출원가×365 DSO = 매출채권/매출×365
DPO = 매입채무/매출원가×365
목표 업종
평균↓ 개선 추세
FCF 전환율 — 이익의 현금화 품질 중기 구현 쉬움
EBITDA가 실제 FCF로 얼마나 전환되는지 비율. 높을수록 회계이익과 실제 현금이 일치함. 70% 미만이면 자본지출·운전자본 부담이 크다는 신호.
FCF 전환율 = FCF / EBITDA → ≥ 70% 권장
기준선 70% 품질 기준
매출 성장 가속도 — Revenue Acceleration 중기 구현 쉬움
이번 분기 매출 성장률이 전기 대비 빨라지고 있는지 측정. 성장주 조기 발굴에 효과적. 분기별 PSR 추이 차트(이미 구현)에 가속도 지표 추가로 확장 가능.
Acc = QoQ매출성장(t) − QoQ매출성장(t-1) → > 0 양수면 가속

배당 성장 팩터 — Dividend Growth Rate 단기 구현 쉬움
배당금을 지속적으로 늘리는 기업 선별. 5년 배당 CAGR ≥ 5%이면서 배당 삭감 이력 없음. 배당귀족 스타일. yfinance dividends 데이터로 즉시 계산 가능.
DGR = (배당(t) / 배당(t-5)) ^ (1/5) − 1 → ≥ 5% AND 무삭감
조건 5yr CAGR ≥ 5%
SUE / PEAD — 실적 서프라이즈 모멘텀 중기 구현 중간
Standardized Unexpected Earnings. 실제 EPS가 컨센서스 대비 얼마나 초과했는지 정규화. PEAD(Post-Earnings-Announcement Drift)는 서프라이즈 후 60일간 주가 드리프트 현상. 국내에서는 증권사 컨센서스 데이터 소스 확보가 과제.
SUE = (실제EPS − 예상EPS) / σ(EPS 추정 오차)
PEAD: SUE > 0 종목 → 발표 후 60일 보유
드리프트 60일 Bernard 1989
AQR Quality Minus Junk (QMJ) 중기 구현 복잡
수익성·성장성·안전성·주주환원 4개 축, 총 15개 지표 조합. 고품질 롱 + 저품질 숏 팩터. 가장 정교한 퀄리티 팩터이나 지표 수집이 복잡. 단계적으로 축을 하나씩 추가하는 전략 권장.
Quality = z(수익성) + z(성장성) + z(안전성) + z(주주환원)
수익성: ROE·ROA·ROC·GP/A·현금ROA·발생주의
성장성: △ROE·△ROA·△GP/A·△OCF
안전성: BAB베타·레버리지·Z-Score·O-Score
지표 수 15개 4개 축
Alpha Architect QMOM + Frog-in-Pan 중기 구현 복잡
Frog-in-the-Pan(FIP)은 모멘텀의 질을 측정. 큰 폭의 점프 없이 조용히 꾸준히 오른 모멘텀이 더 지속성이 있다는 이론. 12개월 중 매주 소폭 상승 = 고품질 모멘텀. 세력 탐지와 조합 가능.
FIP = 1 − (음수 수익일수 / 총 거래일수)
높을수록 꾸준한 상승 모멘텀 (1에 가까울수록 좋음)
FIP 목표 ≥ 0.7 꾸준한 상승

BAB — Betting Against Beta (저베타 팩터) 장기 구현 복잡
저베타 주식이 CAPM 예측을 초과하는 수익을 낸다는 이상현상. AQR Frazzini & Pedersen 2014. 레버리지 제약이 있는 기관투자자가 고베타를 과매수하는 구조적 원인. 하락방어 탭과 연계 가능.
베타 = Cov(종목수익, 시장수익) / Var(시장수익)
BAB: 저베타(β<0.5) 롱 / 고베타(β>1.5) 숏
이상현상 저베타
초과수익 AQR 2014
MAX 이상현상 — 최대 일일수익률 역방향 장기 구현 쉬움
지난 달 하루 최대 수익률이 컸던 주식을 회피하는 전략. 복권 선호 행동 때문에 개인투자자가 과매수한 주식이 이후 저조한 수익을 내는 현상. 스캔 제외 필터로 활용 가능.
MAX = max(일일수익률, 1개월) → 하위 종목 선호
