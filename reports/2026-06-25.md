- `get_tracker_summary(picks)` — 승률·평균수익률·최고/최저 종목 요약 dict

### 4. 리포트 탭 '📂 열기' 버튼 수정

- 원인: `file://` URL은 브라우저 보안 정책상 HTTP 페이지(localhost:3000)에서 클릭으로 열리지 않음
- 해결: `rx.link(href="file:///...")` → `rx.button(on_click=State.open_report_file(filepath))`
- `open_report_file(filepath)` 이벤트 핸들러 추가: `os.startfile(path)` 호출 (Windows 기본 브라우저로 열기)
- `load_report_files()`에서 `file_url` 대신 `filepath` (절대 경로) 저장
