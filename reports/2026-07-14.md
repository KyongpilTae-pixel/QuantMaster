## 작업 요약

### 1. 역발상 과매도 스캔 추가 (`mean_reversion`)

**전략**: RSI14 < 임계값 AND 종가 < 볼린저밴드 하단(20일, 2σ) — 공포 구간 역발상 매수 후보 발굴

**구현 파일**: `utils/mean_reversion_scanner.py` (신규)
- `_calc_mr(args)`: 단일 종목 과매도 조건 계산
  - RSI14: `delta.clip().rolling(14).mean()`, avg_loss==0 → RSI=100
  - BB(20일, 2σ): `rolling(20).std(ddof=1)`
  - 복합 점수: RSI 컴포넌트(최대 50) + BB 이탈 폭 컴포넌트(최대 50) = 100점 만점
  - 부가지표: 거래량비(5d/20d), 1W/1M 수익률
- `scan_mean_reversion(market, min_mktcap_eok, max_rsi, top_n, max_universe, _timeout_s)`:
  - KR: `fetch_kr_stock_listing` → 시총 필터 → 상위 N개
  - US: `fdr.StockListing("S&P500")` → 상위 N개
  - `ThreadPoolExecutor(max_workers=10)` 병렬 처리
  - bool 플래그 미리 계산 (`rsi_extreme`, `vol_up`, `score_high` 등)

**`utils/indicators.py`** 확장:
- `TechnicalIndicators.calc_rsi(closes, period=14)`: RSI 시리즈 반환
- `TechnicalIndicators.calc_bb(closes, period=20, std_mult=2.0)`: (upper, mid, lower) 튜플 반환

**`main/main.py`** 추가:
- State: `mean_reversion_results`, `mr_rsi_max: List[float] = [30.0]`, `mr_min_mktcap: int = 3_000`
- `set_mr_rsi_max()`, `set_mr_min_mktcap()` setter
- `run_scan()` 분기: `scan_mode == "mean_reversion"` → 0.5s polling
- `sidebar_controls()`: RSI14 슬라이더(15~40) + 시총 버튼 옵션 + Callout 설명
- `mean_reversion_table()`: 순위/종목명/RSI14/BB괴리/1W/1M/거래량비/점수/현재가/시총/[조회]
- `scanner_tab()`: `mean_reversion_table()` cond 추가

### 4. Piotroski F-Score — 다중 팩터 모델 Phase 1

**`utils/factor_loader.py`** (신규):
- `load_f_score(symbol, market)`: 9점 만점 재무 건전성 점수 계산
  - yfinance `financials` / `cashflow` / `balance_sheet` 연간 2년치 활용
  - KR 심볼: `.KS`(KOSPI) / `.KQ`(KOSDAQ) 접미사 자동 변환

| 그룹 | # | 기준 | 데이터 소스 |
|------|---|------|------------|
| 수익성 | F1 | ROA > 0 | net_income / total_assets |
| 수익성 | F2 | 영업현금흐름 > 0 | operating_cash_flow |
| 수익성 | F3 | ROA 전년 대비 개선 | ΔROAt > 0 |
| 수익성 | F4 | 발생주의 (OCF/자산 > ROA) | accrual ratio |
| 레버리지 | F5 | 장기부채 비율 감소 | lt_debt / total_assets |
| 레버리지 | F6 | 유동비율 증가 | current_assets / current_liabilities |
| 레버리지 | F7 | 신주 발행 없음 | shares_t ≤ shares_t1 × 1.02 |
| 효율성 | F8 | 매출총이익률 증가 | gross_profit / revenue |
| 효율성 | F9 | 자산회전율 증가 | revenue / total_assets |

**`main/main.py`** 추가:
- State: `f_score_loading: bool`, `f_score_score: int = -1`, `f_score_error: str`, `f_score_criteria: List[dict]`
- `select_stock` (퀀트), `select_stock` (세력), `goto_analysis` — PSR 로드 완료 후 F-Score 비동기 로드
- `f_score_panel()` UI: 점수 배지 (녹색≥7/황색≥5/적색<5) + 9개 기준 체크리스트
- 분석 탭: 가격 차트 위에 F-Score 패널 표시 (f_score_loading 중이거나 점수 유효할 때)
