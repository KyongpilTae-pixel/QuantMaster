## 기술적 지표

### VWAP (Volume-Weighted Average Price)

**정의**: 특정 기간의 (가격 × 거래량) 누적합을 거래량 누적합으로 나눈 거래량 가중 평균가.

```
VWAP_N = Σ(TP_i × Vol_i) / Σ(Vol_i)    (i = 최근 N 거래일)
TP = (High + Low + Close) / 3            (Typical Price)
```

**사용 목적**
- 기관·세력의 매수 기준선 추정.
- 종가 > VWAP → 기관 평균 매수가 상회 → 매수 신호.
- 눌림목 매수의 지지선 기준.

**QuantMaster 기본값**: 120일 VWAP. 눌림목 분할 매수 플랜에서 3차 매수 목표가로 사용.

---

### TWAP (Time-Weighted Average Price)

**정의**: 특정 기간의 Typical Price 단순 평균. VWAP과 달리 거래량을 반영하지 않음.

```
TWAP_N = Σ(TP_i) / N
```

**사용 목적**: VWAP 대비 시간 기준 이동평균 — 거래량 왜곡 없이 가격 추세 파악.

---

### MFI (Money Flow Index)

**정의**: RSI와 유사하지만 거래량을 반영한 0~100 범위의 수급 강도 지표.

```
Positive MF = Σ(TP × Vol)   for TP > prev TP   (14일 합계)
Negative MF = Σ(TP × Vol)   for TP < prev TP
MFI = 100 − 100 / (1 + Positive MF / Negative MF)
```

**해석**
| 값 | 신호 |
|-----|------|
| ≥ 70 | 과매수 (단기 조정 주의) |
| 50~70 | 매수 우위 (스마트 머니 유입) |
| < 50 | 매도 우위 |
| ≤ 30 | 과매도 (반등 기대) |

**QuantMaster 엣지 케이스**: 순수 상승 추세(Negative MF=0) → MFI=100, 횡보(양쪽 0) → MFI=50.

---

### OBV (On-Balance Volume)

**정의**: 거래량에 방향성을 부여해 누적한 지표. 종가 상승일엔 거래량을 더하고, 하락일엔 뺀다.

```
OBV_t = OBV_{t-1} + Vol_t   (Close_t > Close_{t-1})
OBV_t = OBV_{t-1} - Vol_t   (Close_t < Close_{t-1})
```

**신호**: OBV > OBV_Signal(20일 MA) → 세력 매집 가능성 (거래량 유입 선행).

---

### RSI (Relative Strength Index)

**정의**: 14일간 상승폭 평균과 하락폭 평균의 비율로 계산한 0~100 범위의 모멘텀 지표.

```
RS = Avg(Gain, 14) / Avg(Loss, 14)
RSI = 100 − 100 / (1 + RS)
```

**해석**
| 값 | 신호 |
|-----|------|
| ≥ 70 | 과열 (단기 매도 압력) |
| 30~70 | 중립 |
| ≤ 30 | 과매도 (반등 가능성) |

**QuantMaster 활용**: 가격 차트 하단 서브차트로 표시. 70/30 기준선 함께 표시.

---

### Bollinger Bands (볼린저 밴드)

**정의**: 20일 이동평균(중심선)을 기준으로 ±2σ(표준편차)를 더하고 뺀 상한/하한 밴드.

```
BB_mid   = SMA(Close, 20)
BB_upper = BB_mid + 2 × StdDev(Close, 20)
BB_lower = BB_mid − 2 × StdDev(Close, 20)
```

**해석**
- 종가가 BB_upper 돌파 → 과열 또는 강한 상승 추세.
- 종가가 BB_lower 하락 → 과매도 또는 하락 추세.
- 밴드 폭이 좁아질수록 변동성 수축 → 큰 방향 이탈 준비.

**QuantMaster 활용**: 가격 차트에 BB 상단/하단 점선으로 오버레이.

---

### ATR (Average True Range)

**정의**: 14일간의 True Range 평균. True Range는 다음 세 값의 최댓값.

```
TR_t = max(High_t − Low_t,  |High_t − Close_{t-1}|,  |Low_t − Close_{t-1}|)
ATR14 = TR.rolling(14).mean()
```

**활용**: 종목별 변동성을 반영한 손절선 설정. 거래량·가격 수준 무관하게 비교 가능.

## 퀀트 지표

### PBR (Price-to-Book Ratio)

**정의**: 시가총액 / 순자산(장부가). 주가가 자산 대비 몇 배에 거래되는지 표시.

- PBR < 1.0 → 청산가치 이하 거래 (저평가 신호).
- QuantMaster 기본 필터: PBR ≤ 1.2.

---

### PSR (Price-to-Sales Ratio)

**정의**: 시가총액 / 연간 매출액. 매출 기준 밸류에이션.

- 수익이 없는 성장주에서 PER 대신 활용.
- QuantMaster: 분기별 PSR 추이를 바 차트로 표시.

---

### GPA / GP/A (Gross Profitability)

**정의**: (매출총이익) / (총자산). Novy-Marx(2013)가 제안한 수익성 지표.

- 자산 대비 얼마나 효율적으로 이익을 내는지 측정.
- QuantMaster: GPA 백분위 ≥ 60% 기본 필터.

---

### Piotroski F-Score

**정의**: 재무제표에서 9개 항목을 0/1로 채점하는 재무 건전성 점수 (0~9점).

| 그룹 | 기준 |
|------|------|
| 수익성 (4점) | ROA > 0, 영업현금흐름 > 0, ROA 개선, 발생주의 음수 |
| 레버리지/유동성 (3점) | 부채비율 감소, 유동비율 개선, 주식 발행 없음 |
| 효율성 (2점) | 매출총이익률 개선, 자산회전율 개선 |

- ≥ 7점: 재무 건전 (녹색).
- 5~6점: 보통 (황색).
- < 5점: 취약 (적색).

---

### EV/EBITDA

**정의**: 기업가치(EV) / EBITDA. M&A 관점의 밸류에이션 배수.

- EV = 시가총액 + 순부채.
- EBITDA = 영업이익 + 감가상각.
- < 10x: 저평가 (녹색), < 20x: 보통 (황색), ≥ 20x: 고평가 (적색).

---

### P/FCF

**정의**: 시가총액 / 잉여현금흐름(Free Cash Flow). 현금 창출력 대비 주가.

- FCF = 영업현금흐름 − 자본적지출(CapEx).
- < 15x: 저평가 (녹색), < 25x: 보통 (황색), ≥ 25x: 고평가 (적색).

## 스캔 전략

### 52주 신고가

**정의**: 최근 252거래일(약 1년) 종가 최고점 대비 현재 종가가 99.5% 이상인 경우.

```python
high_252 = Close.rolling(252, min_periods=60).max().iloc[-1]
new_high_52w = (Close.iloc[-1] >= high_252 * 0.995)
```

**의미**: 신고가 돌파는 매물 소화 완료 및 추세 가속 신호. 모멘텀 투자의 핵심 진입 조건.
**표시**: 스캔 결과 테이블에 `52W↑` 녹색 배지로 표시.

---

### VKOSPI (한국 공포지수)

**정의**: KOSPI200 옵션의 내재변동성에서 산출하는 한국판 VIX (변동성 지수). 한국거래소(KRX)가 산출·발표.

**해석**
| 값 | 레벨 | 의미 |
|-----|------|------|
| ≥ 25 | 공포 | 시장 불안 극대화. 역발상 매수 기회 가능 |
| 20~25 | 주의 | 변동성 상승 국면. 포지션 관리 필요 |
| < 20 | 안도 | 안정 국면. 추세 추종 유리 |

**QuantMaster 활용**: 분석 탭 KR 종목 선택 시 배너로 표시 (조회 버튼으로 실시간 로드).

## 리스크 관리

### ATR 기반 손절

**공식**: `손절가 = 현재가 − 2.5 × ATR14`

- ATR14가 클수록 손절 폭이 넓어져 변동성이 큰 종목의 섣부른 손절컷 방지.
- ATR14 제공 불가 시 fallback: `VWAP × 0.96` (고정 4% 손절).
- 배수 2.5는 `_ATR_MULT` 상수로 조정 가능.

---

### Kelly Criterion (켈리 기준)

**공식**:
```
f* = (b × p − q) / b

where:
  p = 승률 (0~1)
  q = 1 − p (패율)
  b = 평균 수익률 / 평균 손실률 (수익/손실 비율)
```

**Half-Kelly**: `f_half = f* × 0.5`  
과변동성·추정 오차를 감안해 풀 켈리의 50%만 사용.

**QuantMaster 적용**
- 백테스트 통계(승률·평균수익률·MDD)에서 자동 계산.
- 백테스트 미실행 시 기본값: 승률 55%, 평균수익 8%, 평균손실 4%.
- 최대 한도: 총 예산의 25% (`_KELLY_MAX`).
- 분할 매수 플랜 계산 후 "켈리 권장 투자 비율"과 금액 표시.

---

### 드로우다운 서킷브레이커

**정의**: 포트폴리오 전체 손익률이 특정 임계값 이하로 떨어졌을 때 자동 경보를 발동해 추가 손실을 막는 규칙.

| 임계값 | 경보 레벨 | 권장 조치 |
|--------|---------|---------|
| −10% | 주의 (황색) | 신규 매수 규모 축소, 손절 기준 재검토 |
| −15% | 경고 (주황) | 포지션 50% 축소 |
| −20% | 서킷브레이커 (적색) | 전체 포지션 청산 / 신규 매수 중단 |

**QuantMaster 표시**: 보유종목분석 탭 손익률 카드 하단에 조건부 배너로 표시.

## 백테스트 비용 모델

실제 거래 비용을 반영하지 않으면 백테스트 수익률이 과장된다. QuantMaster는 매수/매도 시 다음 비용을 적용한다.

### 한국 주식 (KR)

| 항목 | 비율 | 적용 방향 |
|------|------|---------|
| 거래세 | 0.18% | 매도만 |
| 수수료 | 0.015% | 양방향 각 |
| 슬리피지 | 0.05% | 양방향 각 |
| **왕복 합계** | **~0.33%** | |

### 미국 주식 (US)

| 항목 | 비율 | 적용 방향 |
|------|------|---------|
| 거래세 | 없음 | — |
| 수수료 | 0.005% | 양방향 각 |
| 슬리피지 | 0.05% | 양방향 각 |
| **왕복 합계** | **~0.11%** | |

**적용 방식**
- 매수 시: `유효 매수가 = 가격 × (1 + buy_cost)` — 살 수 있는 주수 감소.
- 매도 시: `실수령액 = 주수 × 가격 × (1 − sell_cost)`.
- 백테스트 결과 카드에 `거래비용(왕복)%` 별도 표시.

## 포트폴리오 분석

### 손익(PnL) 계산

```
투자금     = 매수가 × 수량
현재 평가액 = 현재가 × 수량
손익금액   = 현재 평가액 − 투자금
손익률(%)  = 손익금액 / 투자금 × 100
```

### 포트폴리오 집계

```
총 투자금  = Σ(개별 투자금)
총 손익    = Σ(개별 손익금액)
전체 손익률 = 총 손익 / 총 투자금 × 100
```

매수가 또는 수량 미입력 종목은 집계에서 제외된다.
